# Divide and Conquer Algorithm Visualization

This project visualizes the Divide and Conquer algorithm using Merge Sort.

## Algorithm Used
Merge Sort

## Divide and Conquer Concept

Divide:
The array is divided into two halves recursively.

Conquer:
Each half is sorted individually.

Combine:
The sorted halves are merged together.

## Recurrence Relation

T(n) = 2T(n/2) + O(n)

Using the Master Theorem:

Time Complexity = O(n log n)
