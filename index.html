Divide and Conquer Explanation

Divide: The array is divided into two halves recursively.

Conquer: Each half is sorted separately.

Combine: The sorted halves are merged together.

Time Complexity

Recurrence Relation: T(n) = 2T(n/2) + O(n)

Using Master Theorem:

Time Complexity = O(n log n)
